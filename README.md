### Vulnerabilities
- Create
When creating a Vulnerability, a Category must be selected (or No Category)

A Vulnerability is defined by:

    Title
    Type
    Language
    Description
    Observation
    CVSS
    Remediation
    Remediation Complexity
    Remediation Priority
    References
    Category
